Micronaut Maven Plugin - Plugin

  • Last Published: 2024-10-24|

mn:generate-openapi-client

Full name:

io.micronaut.maven:micronaut-maven-plugin:4.7.0-SNAPSHOT:generate-openapi-client

Description:

Generates an OpenAPI client. The sources are generated in the target directory.

Attributes:

  • Requires a Maven project to be executed.
  • The goal is not marked as thread-safe and thus does not support parallel builds.
  • Since version: 4.0.0.
  • Binds by default to the lifecycle phase: generate-sources.
